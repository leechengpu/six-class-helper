# 01_法規

## 用途

存放政府採購法本文、施行細則與相關子法，作為整個顧問系統所有判斷的基礎法源依據。本資料夾內容需保持與現行法規同步，採購爭議回應、文件檢查、情境諮詢等功能皆以此為參照來源。

## 應包含內容

- 政府採購法全文（分章節）
- 採購法施行細則
- 最有利標評選辦法
- 異質採購最低標辦法
- 押標金保證金暨其他擔保作業辦法
- 投標廠商資格與特殊或巨額採購認定標準
- 共同投標辦法

## 命名規範

`[法規名稱]_[版本日期].md`

例：`政府採購法_20230614.md`
