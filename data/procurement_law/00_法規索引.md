## 使用建議

### 用 Claude Code 查條文
直接在 Obsidian 或 Claude Code 內搜尋條文關鍵字（例如「第31條」、「押標金」、「最有利標」），定位到對應 Markdown 後再查原文。Markdown 保留 pdftotext 原排版，部分表格可能有空白錯位，請以 PDF 為準。

### 用 Gemini Gem「採購法顧問」
1. 進入 Gem 設定 → 知識庫
2. 把 `~/Downloads/採購法彙編35版_分章/` 下所有 PDF 全數上傳（共 17 份）
3. 上傳順序建議按編號 01 → 13，方便 Gem 依上下文檢索
4. 單檔皆 <= 約 100 MB，符合 Gemini 知識庫單檔上限

### 引用規範
任何從此彙編引用的內容，務必註明「《政府採購法令彙編》第 35 版，第 X 頁」。Obsidian 內部引用可使用 `[[檔名#條號]]` 形式。
